Principais Formatações

Negrito / Destaque

Colocando a tag B a frase fica em negrito

A tag Strong Serve para deixar uma frase destacada

Itálico / ênfase

Nesta frase conseguimos usar a tag I para deixar em italico ou usar o ctrl + shift + p (Não semantica)

Nesta frase usamos a tag EM para Ênfase ou a mesma coisa do ctrl e ela é Semantica

Marca textos

Podemos usar um marca texto usando a tag Mark e colocar o style + background dentro da tag mask

Texto Pequeno

Colocando a tag small as frases ficaram pequenas

Texto Deletados

Podemos marcar um texto como excluído usando a tag Del, para informar que ele deve ser lido, mas não considerado

Texto Inserido

Podemos usar a Tag INS para dar uma enfase e indicar que ele foi adicionado depois

Texto Sobrescrito

Podemos usar a tag SUP para deixar encima

Texto Subriscrito

Usamos a tag SUB para deixar as coisas embaixo
